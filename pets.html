Which row breaks first?

The **letter row** breaks first because it uses `col-sm-2`, meaning each letter takes up a smaller portion of the screen. On smaller screens, Bootstrap forces these elements to wrap earlier compared to the pet names, which use `col-md`, making them more flexible.
